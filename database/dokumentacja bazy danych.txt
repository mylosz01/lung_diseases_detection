Dokumentacja bazy danych:

Recordings:
	RecordingID - id nagrania
	PatientID - id pacjenta którego tyczy nagranie
	Description - opis lekarza odnośnie nagrania
	DoctorDiagnosis - werdykt lekarza odnośnie choroby
	ModelDiagnosis - werdykt modelu co do choroby
	Accuracy - pewność werdyktu modelu kklasyfikującego
	Date - data wykonania nagrania (wrzucenia w aplikację po stronie klienta)
	AudioFile - plik nagrania
	ModifiedDate - data modyfikacji wpisu przechowującego nagranie
	rowguid

Patients:
	PatientID - id pacjenta
	Name - imie
	Surname - nazwisko
	Pesel - nr PESEL
	DoctorID - ID doktora prowadzącego
	Age - wiek pacjenta
	Phone - numer telefonu
	Recomendations - aktualnie przepisane leki (zalecenia lekarza, aktualizowane przez lekarza po każdym nagraniu)
	ModifiedDate - data modyfikacji wpisu o pacjencie
	Rowguid

Doctor:
	doctorID - id lekarza
	Name - imię lekarza
	Surname - nazwisko lekarza
	modifiedDate - data modyfikacji wpisu o lekarzu
